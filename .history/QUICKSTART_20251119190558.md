### üåê Web Application (Better UI + ML Features)
Best for: Better interface, adjustable settings, fuel efficiency predictions

**Step 1 - Start Server:**
```bash
python app.py
```

The server will automatically:
- Load 38,113+ vehicle records from dataset
- Train 3 Random Forest ML models
- Start Flask server on http://localhost:5000

**Step 2 - Open Browser:**
Go to: http://localhost:5000

**Step 3 - Try Drowsiness Detection:**
- Click "Start Detection"
- Close eyes for 5 seconds to test

**Step 4 - Try Fuel Efficiency Predictor:**
- Click "Predict Fuel Efficiency"
- Select sample vehicle or enter custom specs
- Get ML-powered predictions (City/Highway/Combined km/l)
